# NGINX Log Champions

## 1. Nginx Liquefy (Complex)
- **Filename:** `NginxChampion_Complex_Liquefy.py`
- **Best For:** Real-world traffic logs with unique requests and varied metadata.
- **Tech:** Field-aware template extraction.

## 2. Nginx Universal (Repetitive)
- **Filename:** `NginxChampion_Repetitive_Universal.py`
