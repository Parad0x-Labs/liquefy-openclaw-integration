# Syslog Champions

## 1. Syslog Liquefy V2 (Complex)
- **Filename:** `SyslogChampion_Complex_Liquefy_V2.py`
- **Best For:** RFC3164/5424 logs from multiple hosts.
- **Tech:** Global Smart Columns (Dict/Delta/RLE) + Template Extraction.
- **Ratio:** Strong performance on mixed system logs.
- **Features:** Lossless, Searchable.

## 2. Syslog Universal (Repetitive)
- **Filename:** `SyslogChampion_Repetitive_Universal.py`
- **Best For:** Repeating "CRON" or "UPTIME" messages.
- **Tech:** Pattern-ID deduplication.
