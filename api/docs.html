Endpoints

POST /api/compress/log

Synchronous Log Compression

Compresses a raw log stream using one of the 21 specialized engines. Returns a URI to the cryptographically sealed `.null` archive.

Request Body (multipart/form-data)

ParameterTypeDescription
fileFileRaw log file (Max 50MB Lab Limit).
engineStringID of tuned engine (e.g., LIQUEFY_V1_JSON). Defaults to LIQUEFY_V1_MIXED.
tenant_idStringEnterprise tenant identifier for isolated key derivation.

Response (application/json)

{ "status": "success", "job_id": "uuid-v4-identifier", "engine": "LIQUEFY_V1_JSON", "original_size": 197500421, "compressed_size": 5821004, "ratio": "33.92x", "download_url": "/api/archive/filename.null" }
POST /api/decompress

Bit-Perfect Restoration

Restores a `.null` archive to its exact original state. Mandatory Round-Trip Verification (MRTV) ensures zero bit loss.

Request Body (multipart/form-data)

ParameterTypeDescription
fileFileThe .null compressed archive.
tenant_idStringMust match the identifier used during compression.

Response (application/json)

{ "status": "success", "job_id": "uuid-v4-identifier", "download_url": "/api/archive/filename.restored" }
